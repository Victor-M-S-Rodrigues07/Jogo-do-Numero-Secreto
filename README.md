## <h2> 📝 Descrição do Projeto </h2>

Esta aplicação contém o Jogo do Número Secreto, onde o usuário precisa descobrir o número aleatório escolhido pela máquina. O computador fornecerá dicas se o chute realizado é menor ou maior que a resposta. O usuário ganha quando insere o número correto. Ao final, o jogador poderá ver quantas tentativas foram usadas.<br>

O site foi desenvolvido para o curso "Lógica de Programação: Explore Funções e Listas", da Alura. Foram fornecidos o HTML e o CSS para que, com o JavaScript, os elementos da página pudessem ser alterados visando atingir o objetivo do jogo -- sortear um número e aceitar respostas do usuário. <br>

## <h2> 🎯 Objetivos da Aprendizagem </h2>

- Descubra como utilizar listas, uma das principais estruturas de dados no mundo da programação;

## <h2> ⚙ Funcionalidades </h2>

- Campo para adicionar a quantidade de números a serem sorteados, com intervalo entre 1 e 10;
- Realizar quantas tentativas desejar;
- Botão para reiniciar e limpar os campos sem a necessidade de atualizar a página via JavaScript;
- Textos informativos criados pelo JavaScript ao clique dos botões disponíveis na página;
- Dicas para saber se o número é maior ou menor que a tentatica através de comparações, garantindo diversão sem comprometer a experiência.
